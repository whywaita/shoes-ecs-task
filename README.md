# shoes-ecs-task

[myshoes](https://github.com/whywaita/myshoes) provider for [Amazon ECS](https://aws.amazon.com/ecs/).

## Environment values

- "ECS_TASK_CLUSTER"
  - The name of ECS Cluster
- "ECS_TASK_DEFINITION_ARN"
  - The ARN of Task Definition
- "ECS_TASK_SUBNET_ID"
  - The ID of Subnet
- "ECS_TASK_REGION"
  - The name of Region
